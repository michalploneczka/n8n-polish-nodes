- `deactivateWorkflow(id: string): Promise<void>` — PATCH /api/v1/workflows/{id} with body `{ "active": false }`. Wrap in try/catch, best-effort cleanup.
- `deleteWorkflow(id: string): Promise<void>` — DELETE /api/v1/workflows/{id}. Wrap in try/catch, best-effort cleanup.
- `callWebhook(path: string, method?: string, body?: object): Promise<any>` — fetch `${N8N_BASE_URL}/webhook/${path}` with given method (default GET). If body provided, set Content-Type: application/json. Throw on non-ok response including status and response text.
- `createCredential(name: string, typeName: string, data: Record<string, string>): Promise<string>` — POST /api/v1/credentials with body `{ name, type: typeName, data }`. Returns credential id.
- `isNetworkError(error: unknown): boolean` — check if error message contains ECONNREFUSED, ETIMEDOUT, ENOTFOUND, or 'fetch failed'.
- `todayDateString(): string` — returns `new Date().toISOString().split('T')[0]`
- `waitForN8n(baseUrl: string, timeoutMs?: number): Promise<void>` — poll /healthz every 2s until 200 OK or timeout (default 60000ms). Reuse the exact pattern from `__tests__/integration/helpers.ts`.
- `loadFixture(name: string): object` — read and JSON.parse from `__tests__/e2e/fixtures/${name}`

3. Create `scripts/e2e-test.sh`:
```bash
#!/usr/bin/env bash
set -euo pipefail

COMPOSE_FILE="docker-compose.test.yml"
PROJECT_NAME="n8n-e2e-test"

cleanup() {
  echo "Tearing down test containers..."
  docker compose -f "$COMPOSE_FILE" -p "$PROJECT_NAME" down --volumes --remove-orphans 2>/dev/null || true
}
trap cleanup EXIT

echo "Building all packages..."
pnpm run build:all

echo "Starting n8n test container..."
docker compose -f "$COMPOSE_FILE" -p "$PROJECT_NAME" up -d --wait

echo "Running E2E tests..."
CEIDG_API_KEY="${CEIDG_API_KEY:-}" npx jest --config jest.config.e2e.js "$@"

echo "E2E tests passed!"
```
Make the script executable: `chmod +x scripts/e2e-test.sh`
