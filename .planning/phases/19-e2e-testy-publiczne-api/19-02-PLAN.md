**Error handling pattern for ALL test bodies:**
```typescript

  try {
    const result = await callWebhook('e2e-xxx');
    // assertions...

    if (isNetworkError(error)) {
      console.warn('API unavailable, skipping E2E test');
      return;
    }
    throw error;
  }

```

**Important implementation notes:**
- n8n wraps node output in items. The webhook with responseMode: lastNode returns the raw output of the last node. For declarative nodes, this is typically the JSON body from the API response. The exact wrapping depends on n8n version -- be prepared for the result to be the direct API response OR wrapped in `[{ json: { ... } }]`. Add a helper at the top of the file:
```typescript
function unwrapResult(raw: any): any {
  // n8n may return array of items or single item
  if (Array.isArray(raw) && raw.length > 0) {
    return raw[0].json ?? raw[0];
  }
  return raw?.json ?? raw;
}
```
Use `unwrapResult()` on every `callWebhook` response before asserting.
