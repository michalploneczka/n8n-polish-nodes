**2. scripts/integration-test.sh** (chmod +x):
```bash
#!/usr/bin/env bash
set -euo pipefail

COMPOSE_FILE="docker-compose.test.yml"
PROJECT_NAME="n8n-integration-test"

cleanup() {
  echo "Tearing down test containers..."
  docker compose -f "$COMPOSE_FILE" -p "$PROJECT_NAME" down --volumes --remove-orphans 2>/dev/null || true
}
trap cleanup EXIT

echo "Building all packages..."
pnpm run build:all

echo "Starting n8n test container..."
docker compose -f "$COMPOSE_FILE" -p "$PROJECT_NAME" up -d --wait

echo "Running integration tests..."
npx jest --config jest.config.integration.js

echo "Integration tests passed!"
```

**2. Interface and function for expected nodes:**
```typescript
interface ExpectedNode {
  packageName: string;   // e.g. "n8n-nodes-smsapi"
  nodeTypes: string[];   // e.g. ["n8n-nodes-smsapi.smsapi"] — from n8n.nodes paths
  credentialTypes: string[]; // e.g. ["smsapiApi"] — from n8n.credentials paths
}
```

`getExpectedNodes(): ExpectedNode[]`
- Reads `packages/` directory with `fs.readdirSync`
- Filters to directories starting with `n8n-nodes-`
- For each, reads `package.json` and extracts from the `n8n` section:
  - `nodeTypes`: For each entry in `n8n.nodes` array, extract class name from path like `dist/nodes/Smsapi/Smsapi.node.js` -> class name `Smsapi` -> node type `${packageName}.${className.charAt(0).toLowerCase() + className.slice(1)}` giving `n8n-nodes-smsapi.smsapi`
  - `credentialTypes`: For each entry in `n8n.credentials` array (may be empty/undefined), extract class name from path like `dist/credentials/SmsapiApi.credentials.js` -> credential name `smsapiApi` (first char lowercase)
- Returns sorted by packageName

```typescript
export const N8N_BASE_URL = 'http://localhost:5679';
export const N8N_INTERNAL_TYPES_URL = `${N8N_BASE_URL}/types/nodes.json`;
```

Important: The node type derivation MUST handle edge cases in actual package.json paths. Read at least 4 different package.json files to verify the path patterns before implementing the extraction logic. Some packages might have multiple nodes or credentials.
