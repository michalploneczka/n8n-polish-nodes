3. **Search by Name fields** (displayOptions show operation=searchByName):
   - `nazwa` (string, required, description: 'Company name or part of name to search for')

4. **Get by ID fields** (displayOptions show operation=get):
   - `companyId` (string, required, description: 'CEIDG entry ID')

```typescript
  displayOptions: { show: { operation: ['searchByNip'] } },

    request: { method: 'GET', url: '/firmy' },

  },
}
```

```typescript
{
```
