    ```typescript

      this: IExecuteFunctions,
      method: IHttpRequestMethods,
      endpoint: string,
      body: IDataObject = {},
      qs: IDataObject = {},
      options: IDataObject = {},
    ): Promise<any>
    ```
    Implementation:
    - Call `await this.getCredentials('fakturowniaApi')` to get `apiToken` and `subdomain`
    - Strip `.fakturownia.pl` suffix from subdomain if user accidentally included it: `subdomain.replace(/\.fakturownia\.pl$/i, '')`
    - Construct URL: `https://${cleanSubdomain}.fakturownia.pl${endpoint}`
    - Add `api_token: apiToken` to qs (query parameter auth, NOT header auth)
    - Build `IHttpRequestOptions` with method, url, qs, json: true
    - If method is POST or PUT, set `body` on request options
    - Spread `...options` into request options (for encoding: 'arraybuffer' override on PDF)
    - Call `return this.helpers.httpRequest(requestOptions)`
    - Wrap in try/catch: on error, throw `new NodeApiError(this.getNode(), error as JsonObject, { message: error.message })`

    **2. `fakturowniaApiRequestAllItems`**
    ```typescript
    export async function fakturowniaApiRequestAllItems(
      this: IExecuteFunctions,
      method: IHttpRequestMethods,
      endpoint: string,
      body: IDataObject = {},
      qs: IDataObject = {},
    ): Promise<IDataObject[]>
    ```
    Implementation:
    - `const returnAll = this.getNodeParameter('returnAll', 0) as boolean`
    - `const limit = returnAll ? 0 : (this.getNodeParameter('limit', 0) as number)`
    - `const perPage = 100` (max allowed by Fakturownia)
    - Loop: `let page = 1; const allItems: IDataObject[] = []`
    - Each iteration: call `fakturowniaApiRequest.call(this, method, endpoint, body, { ...qs, page, per_page: perPage })`
    - Push response items into allItems
    - If `!returnAll && allItems.length >= limit`: return `allItems.slice(0, limit)`
    - If `response.length < perPage`: break (last page detected)
    - Increment page
    - Return allItems

    Import types from 'n8n-workflow': `IDataObject`, `IExecuteFunctions`, `IHttpRequestMethods`, `IHttpRequestOptions`, `NodeApiError`, `JsonObject`.
