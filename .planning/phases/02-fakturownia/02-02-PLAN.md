    Create Fakturownia.node.ts as a programmatic node with execute() method.

    **Node description:**
    - `displayName`: 'Fakturownia'
    - `name`: 'fakturownia'
    - `icon`: 'file:../../icons/fakturownia.svg'
    - `group`: ['output']
    - `version`: 1
    - `subtitle`: '={{$parameter["operation"] + ": " + $parameter["resource"]}}'
    - `description`: 'Manage invoices, clients, and products via Fakturownia.pl'
    - `defaults`: { name: 'Fakturownia' }
    - `usableAsTool`: true
    - `inputs`: [NodeConnectionTypes.Main]
    - `outputs`: [NodeConnectionTypes.Main]
    - `credentials`: [{ name: 'fakturowniaApi', required: true }]
    - `properties`: Resource selector with ONLY invoice option for now + spread invoiceOperations, invoiceFields. Client and product options will be added to the resource selector in Plan 03 when their descriptions exist.

    **IMPORTANT -- Resource selector must ONLY include 'invoice' for now:**
    ```typescript
    {
      displayName: 'Resource',
      name: 'resource',
      type: 'options',
      noDataExpression: true,
      options: [
        { name: 'Invoice', value: 'invoice' },
      ],
      default: 'invoice',
    }
    ```
    Do NOT add 'client' or 'product' options here -- they will be added in Plan 03 Task 2 when their resource descriptions are imported. Adding them now would create UI options with no backing field descriptions.

    **execute() method** -- loop over `this.getInputData()` items, for each item:
    - Read `resource` and `operation` from parameters at index `i`
    - Invoice operations dispatch:

    **invoice + list:**
    - Get filters from `this.getNodeParameter('filters', i, {})` as IDataObject
    - Build qs from filters (period, date_from, date_to, kind)
    - Call `fakturowniaApiRequestAllItems.call(this, 'GET', '/invoices.json', {}, qs)`
    - Push each item as `{ json: item }`

    **invoice + get:**
    - `const invoiceId = this.getNodeParameter('invoiceId', i)`
    - Call `fakturowniaApiRequest.call(this, 'GET', \`/invoices/${invoiceId}.json\`)`
    - Push `{ json: response }`

    **invoice + create:**
    - Build invoice body object:
      - `kind`: from parameter
      - `buyer_name`: from parameter
      - `buyer_tax_no`: from parameter
      - `positions`: `JSON.parse(this.getNodeParameter('positions', i) as string)` wrapped in try/catch -- on parse error throw `new NodeApiError(this.getNode(), {}, { message: 'Invalid JSON in Positions field. Expected array of objects.' })`
      - Spread additionalFields (number, sell_date, issue_date, payment_to, payment_type, currency, lang, description)
    - Call `fakturowniaApiRequest.call(this, 'POST', '/invoices.json', { invoice: invoiceBody })`
    - Push `{ json: response }`

    **invoice + update:**
    - `const invoiceId = this.getNodeParameter('invoiceId', i)`
    - Build update body from updateFields collection
    - If updateFields contains `positions`, JSON.parse it (same try/catch pattern)
    - Call `fakturowniaApiRequest.call(this, 'PUT', \`/invoices/${invoiceId}.json\`, { invoice: updateBody })`
    - Push `{ json: response }`

    **invoice + delete:**
    - `const invoiceId = this.getNodeParameter('invoiceId', i)`
    - Call `fakturowniaApiRequest.call(this, 'DELETE', \`/invoices/${invoiceId}.json\`)`
    - Push `{ json: { success: true, id: invoiceId } }`

    **invoice + sendByEmail:**
    - `const invoiceId = this.getNodeParameter('invoiceId', i)`
    - `const emailTo = this.getNodeParameter('emailTo', i) as string`
    - Get emailOptions from `this.getNodeParameter('emailOptions', i, {})` as IDataObject
    - Build body: `{ email_to: emailTo, ...emailOptions }`
    - Call `fakturowniaApiRequest.call(this, 'POST', \`/invoices/${invoiceId}/send_by_email.json\`, body)`
    - Push `{ json: response }`

    **invoice + downloadPdf:**
    - `const invoiceId = this.getNodeParameter('invoiceId', i)`
    - Call `fakturowniaApiRequest.call(this, 'GET', \`/invoices/${invoiceId}.pdf\`, {}, {}, { encoding: 'arraybuffer', returnFullResponse: false })`
    - `const binaryData = await this.helpers.prepareBinaryData(Buffer.from(response as ArrayBuffer), \`invoice-${invoiceId}.pdf\`, 'application/pdf')`
    - Push `{ json: { invoiceId }, binary: { data: binaryData } }`

    **Fallback for unrecognized resource/operation** -- After all if/else branches, add:
    ```typescript
    throw new NodeApiError(this.getNode(), {}, {
      message: `Unsupported resource/operation: ${resource}/${operation}`,
    });
    ```
    This prevents silent empty results for unhandled combinations.

    **Error handling** -- wrap the entire per-item logic in try/catch. On error: if `this.continueOnFail()`, push `{ json: { error: (error as Error).message } }`, else throw.

    Import from resources/invoices: invoiceOperations, invoiceFields
